Building an app for tracking price per usage with a paid membership. 

The motivation behind this is creating a simple yet modern way of keeping track of how often 
I personally visit a Tampere city swimming hall with a paid yearly ticket. By tracking price per visit,
I feel motivated to get my money's worth. 

Implemented with TypeScript + React Native -library. Current demo running with Expo + Expo Go on device. 

###For running the demo:
Install Expo Go from Google Play or App Store.
``` 
git clone git@github.com:GifuSalmiakki/SwimmingApp.git
cd SwimmingApp
npx expo start
```
You'll get a QR-code and a local IP-address port, either of which can be used to see the running code in 
Expo Go -app.
